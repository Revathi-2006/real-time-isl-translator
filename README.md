# Real-Time ISL Video Call Communication System

## Project Overview

The Real-Time ISL Video Call Communication System is a web-based application designed to facilitate communication between hearing-impaired individuals and non-sign language users. The system uses computer vision and machine learning techniques to recognize Indian Sign Language (ISL) gestures in real time and convert them into text during live communication.

The application also supports text-to-sign conversion, sign-to-text conversion, speech-to-text functionality, and real-time video communication.


## Features

* Real-time Indian Sign Language recognition
* Static and dynamic gesture detection
* Live video call communication
* Sign-to-text conversion
* Text-to-sign playback
* Speech-to-text integration
* Real-time chat system
* Communication history

## System Workflow

1. User performs ISL gestures through webcam.
2. MediaPipe extracts hand landmarks.
3. Machine learning models classify the gesture.
4. Recognized sign is converted into text.
5. Text is displayed in the communication interface.
6. Speech and text can be converted back into sign representations.
